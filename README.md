## Анализ Предметной Области (ПО) Futrat

### 1.  Ключевые Сущности и Их Атрибуты

Для полноценной работы приложения Futrat, включая авторизацию, необходимы следующие основные сущности:

| **Сущность**                | **Описание**                                                  | **Основные Атрибуты**                                                                                               |
| --------------------------- | ------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| **1. Пользователь (User)**  | Основная сущность для персонализации и интерактивных функций. | ID, Имя/Никнейм, Email, Пароль (хэшированный), Дата регистрации, **Избранное** (Команды, Игроки, Турниры - ссылки). |
| **2. Турнир (Tournament)**  | Соревнование, по которому собирается статистика.              | ID, Название, Страна, Тип (Лига/Кубок), Текущий Сезон.                                                              |
| **3. Команда (Team)**       | Участник турниров.                                            | ID, Название, Эмблема, Страна, Главный тренер, Состав (список ID Игроков).                                          |
| **4. Игрок (Player)**       | Объект сбора детальной статистики и рейтингов.                | ID, ФИО, Позиция, Гражданство, Текущая Команда (ID), **Статистика** (Голы, Ассисты, Карточки).                      |
| **5. Матч (Match)**         | Основная единица для результатов и прогнозов.                 | ID, Дата/Время, Турнир (ID), Хозяева (ID), Гости (ID), Счет, Статус, **События Матча** (Голы, Замены и т.д.).       |
| **6. Прогноз (Prediction)** | Сущность для интерактивного модуля "Прогнозы пользователей".  | ID, **Пользователь (ID)**, **Матч (ID)**, Прогнозируемый Результат, Набранные Очки (за этот прогноз).               |

---

### 2. Модуль Авторизации и Персонализации

Это базовый модуль, который обеспечивает индивидуальный опыт использования.

#### 1. Управление Пользователями

- **Регистрация/Вход:** Стандартные механизмы с использованием email/пароля или социальных сетей.
- **Профиль:** Возможность редактировать никнейм и настройки уведомлений.


#### 2. Персонализация (Модуль "Избранное")

- **Механика:** Пользователь добавляет в избранное **Команды, Турниры, Игроков**.    
- **Отображение:** Главная страница **динамически** формируется на основе "Избранного" (например, расписание ближайших матчей избранных команд, текущие таблицы избранных турниров).
- **Уведомления:** **Push-уведомления** привязаны к избранным объектам (Голы избранных команд, начало матча и т.д.).    

---

### 3. Основной Информационный Модуль (Статистика)

Этот модуль отвечает за предоставление всей необходимой футбольной информации.

#### 1. Статистика Игроков и Команд

- **Игроки:** Детальные показатели (голы, ассисты, xG, xA, сыгранные минуты) по **Турнирам** и **Сезонам**.
- **Команды:** Турнирные показатели (победы, поражения, очки) + продвинутая статистика (xG, xGA).

#### 2. Турнирные Таблицы и Сетки

- **Таблицы (Лиги):** Стандартное отображение с **зонированием** (ЛЧ, ЛЕ, Вылет).
- **Сетки Плей-офф (Кубки):** Визуальное представление прогресса (например, для Лиги Чемпионов или Чемпионата Мира).

#### 3. Система Рейтингов Futrat

- **Рейтинги Игроков (Индивидуальные):** Топ-10/20 по Голам, Ассистам, Гол+Пас.
- **Рейтинг на "Золотой мяч":** Берется статистика с различных интернет-порталов, откуда собирается собственный рейтинг игроков.

### 4. Модуль Анализа и Прогнозов

Предоставление контекстной информации для повышения вовлеченности и аналитики.

| **Функционал**             | **Требуемые Данные**                             | **Ценность для Пользователя**                               |
| -------------------------- | ------------------------------------------------ | ----------------------------------------------------------- |
| **Форма Команд**           | Последние 5 **Матчей** команды.                  | Оценка текущей готовности.                                  |
| **Личные Встречи (H2H)**   | История **Матчей** между двумя соперниками.      | Понимание исторического преимущества.                       |
| **Продвинутая Статистика** | **xG (Ожидаемые голы), xA (Ожидаемые ассисты)**. | Оценка реального качества игры, а не только результата.     |

---

### 5. Интерактивный Модуль

Этот модуль использует сущность **Пользователь** и **Прогноз** для создания соревновательного элемента.

#### 1. Прогнозы Пользователей (Prediction Game)

- **Ввод:** Пользователь делает **Прогноз** (Исход матча) на конкретный **Матч**.
- **Система Очков:** Начисление очков после завершения матча.
- **Рейтинг Прогнозистов:** **Глобальная таблица лидеров** на основе накопленных очков **Пользователя**.

#### 2. Голосования

- **Команда Недели:** Пользователи могут голосовать за игроков, формируя **символическую сборную** недели/месяца.
- **Лучший Игрок Матча:** Голосование по окончании каждого матча.
